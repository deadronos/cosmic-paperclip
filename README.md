# The Cosmic Paperclip (React + Vite + TypeScript)

Cold, efficient, minimalist incremental game: convert all available matter into paperclips.

## Quick start

1. Install dependencies
   - `npm install`
2. Run dev server
   - `npm run dev`

## Notes

- Save data is stored in `localStorage` (`cosmic-paperclip:save:v1`).
- UI uses Tailwind + shadcn/ui-style components under `src/components/ui`.
